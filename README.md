-   create python virtual env
    ```
     py -m venv .venv

    ```
- Activate venv on windows with git bash or cmd | remove "source" for cmd.

    ```
    source .venv\\Scripts\\activate
    ```

-   install the requirements.txt
    ```
    pip install -r requirements.txt

    ```
